# Render Ray

A rendering engine built for realistic scenes.
NOT built for real-time applications like games; better built for generating images and videos.

## Features

- Rendering scenes from mesh data
- Supports OBJ file format
- Basic lighting and shading models
- Easy-to-use API for scene creation
- Built-in scripting API for editing scenes during rendering
- Built-in scripts for things such as boids and other simulations
- Etc.
